# Band Baaja Hub – Test Plan

## Project Overview
Band Baaja Hub is a wedding planning platform that provides venue booking, catering, band selection, and other services in one place. The project is built using PHP, MySQL (phpMyAdmin), HTML, CSS, and JavaScript.

---

## Objectives
- To verify that all modules function as expected.
- To ensure user-friendly navigation and responsive design.
- To validate data integrity and booking accuracy.

---

## Scope
Modules Covered:
1. User Registration and Login
2. Venue Management
3. Catering and Band Services
4. Booking System
5. Admin Panel
6. Search and Filtering
7. Notifications

---

## Testing Types
| Type | Description |
|------|--------------|
| Unit Testing | Testing individual PHP functions and database queries |
| Integration Testing | Testing interactions between modules (e.g., login → booking) |
| UI / UX Testing | Ensuring proper layout, responsiveness, and accessibility |
| System Testing | Checking the whole system workflow |
| Regression Testing | Re-testing after fixing bugs |

---

## Test Environment
- **Server:** XAMPP (Apache + PHP 8.x)
- **Database:** MySQL (phpMyAdmin)
- **Browser:** Chrome / Edge
- **OS:** Windows 10
